.. _loadlayer:

Load layer into project
=======================
Loads a layer to the current project.

Parameters
..........

``Layer`` [multipleinput]
  Layer to load in the legend

``Loaded layer name`` [string]
  Name of the loaded layer


.. _renamelayer:

Rename layer
============
Renames a layer in the legend.

Parameters
..........

``Layers`` [multipleinput]
  Layer to rename

``New name`` [string]
  Name of the layer
