|MOD| Online Resources

When considering data sources for a map, there is no need to be restricted by
data you need to download onto the computer you're working on. There are online
data sources that you can load data from as long as you are connected to the
Internet.

In this module, we'll cover two kinds of web-based GIS services: Web Mapping
Services (WMS) and Web Feature Services (WFS).
