magnetic_declination
....................

Returns the declination of a magnetic field at a point, in degrees east of north

.. list-table::
   :widths: 15 85

   * - Syntax
     - magnetic_declination(model_name, date, latitude, longitude, height, [model_path])

       [] marks optional arguments
   * - Arguments
     - * **model_name** - Name of magnetic model to use (eg 'wmm2025'). This must match a magnetic model installed on the system.
       * **date** - datetime at which to evaluate the magnetic model
       * **latitude** - latitude of the point at which to evaluate the magnetic model. Should be specified as a WGS84 coordinate.
       * **longitude** - longitude of the point at which to evaluate the magnetic model. Should be specified as a WGS84 coordinate.
       * **height** - height of the point above the ellipsoid (in meters)
       * **model_path** - optional directory path containing the magnetic model. Can be used to reference magnetic models stored outside the default locations.
   * - Examples
     - * ``magnetic_declination('wmm2025', make_datetime(2026,7,1,12,0,0), -35, 138, 0)`` → 7.873899


.. end_magnetic_declination_section
